### Steps to Run
1. **Install the Required Dependencies:**
   ```bash
   pip install -r requirements.txt
   ```
2. **Run the Project:**
   ```bash
   python src/main.py
   ```

### Project Files Description
- **data/**: It contains raw and processed datasets.
- **src/main.py**: Execute all the stages of the project.
- **src/data_loader.py**: Functions for loading and saving the dataset.
- **src/preprocessing.py**: Data cleaning and turning encoded categorical featuers' categories into their real values.
- **src/analysis.py**: Exploratory Data Analysis (EDA) and plotting.
- **src/feature_engineering.py**: Feature selection/reduction and performing one-hot encoding.
- **src/modeling.py**: Model training and evaluation.
- **report/**: It contains the project report.
- **outputs/**: Generated figures and tables.
